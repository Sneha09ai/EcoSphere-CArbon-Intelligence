# EcoSphere Carbon Intelligence Platform

## Overview

EcoSphere Carbon Intelligence Platform is a web-based sustainability application designed to help individuals understand, track, and reduce their environmental impact. The platform calculates an estimated annual carbon footprint based on key lifestyle factors including electricity consumption, transportation habits, dietary choices, and waste generation.

Beyond simple calculations, EcoSphere provides users with actionable insights, visual analytics, historical footprint tracking, and carbon reduction simulations to encourage environmentally responsible decision-making.

## Problem Statement

Many individuals are unaware of how their daily activities contribute to greenhouse gas emissions. Without clear visibility into their environmental impact, it becomes difficult to adopt sustainable habits and make informed choices.

EcoSphere addresses this challenge by transforming everyday lifestyle data into meaningful carbon footprint metrics and providing personalized recommendations for reducing emissions.

## Key Features

### Carbon Footprint Assessment

* Calculates annual carbon emissions (kg CO₂e/year)
* Evaluates multiple lifestyle factors
* Provides detailed emissions breakdown

### Interactive Dashboard

* Displays total environmental impact
* Presents category-wise emission analysis
* Offers a clean and user-friendly interface

### Emission Breakdown Visualization

* Interactive pie chart using Chart.js
* Visual comparison of emission sources
* Easy-to-understand analytics

### Personalized Sustainability Recommendations

* Identifies the highest emission contributor
* Generates tailored suggestions for improvement
* Encourages sustainable behavioral changes

### Historical Tracking

* Stores previous assessments using Local Storage
* Enables users to monitor environmental progress
* Supports long-term sustainability awareness

### Carbon Reduction Simulator

* Estimates potential reductions from eco-friendly actions
* Demonstrates environmental benefits of lifestyle changes
* Motivates users to adopt greener habits

### Accessibility & User Experience

* Responsive design for desktop and mobile devices
* Keyboard-friendly navigation
* Accessible form controls and focus indicators

## Technologies Used
